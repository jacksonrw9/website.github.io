Edge Puck

Purpose Built for Cleanrooms

A self-contained, cleanable gateway with a built-in computer that connects analytical scientific instruments to Digital Systems while meeting strict cleanroom standards — something no other device could deliver until now.

Request a Demo

How the Only Cleanroom-Certified Computer Inside a Gateway Works:

Phizzle's Connected Plant Runs on the Computer and the Gateway Connects to Cleanroom Instruments

Step 1: Connect

The Edge Puck connects to analytical scientific instruments and captures the raw sample data.

Step 2: Automate

Connected Plant standardizes the data using Phizzle's Data Formatting Engine, and enables laboratory technician and quality managers to review results with compliant e-signatures and maintain detailed audit logs for FDA readiness from one UI — all with zero paper trail.

Step 3: Integrate

We integrate Connected Plant into existing infrastructure and build around current SOPs, enabling manufacturing facilities to securely synchronize data to Digital Systems (LIMS, MES, QMS, etc.) without needing to change any processes.

Benefits of the Edge Puck:

Versatile Connectivity

Allows laboratory technicians to push a cart with an APC or a microbial air sampler and take samples throughout a facility while maintaining connectivity. Solves connectivity challenges in cramped cleanroom spaces where conventional gateways are too large or impractical to install.

Secures Vulnerable Analytical Instruments

Analytical instruments often use exposed communication protocols that create network security vulnerabilities. Phizzle's Edge Puck acts as a security barrier, converting vulnerable protocols like Modbus into encrypted, secure communications with MQTT. Eliminating direct network access to your instruments while maintaining full connectivity to your backend systems.

Malware Hardened

The Edge Puck's Read-Only File System provides inherent malware protection — since nothing can be written to the operating system, malicious software simply cannot install or execute.

Edge Puck Use-Cases:

Building Automations - RFID Tracking:

Monitor cleanroom capacity in real-time with automated alerts when thresholds are exceeded. Maintain detailed audit trails for continuous FDA inspection readiness.

AI/ML:

Feed formatted instrument data into machine learning models for predictive maintenance, process optimization, and automated anomaly detection across manufacturing operations.
